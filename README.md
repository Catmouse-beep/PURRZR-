# purrzr
安装教程：
1、不支持虚拟主机，需服务器
2、php>=7.2，MySQL版本>=5.6
3、下载源码，然后上传至你的服务器
3、设置运行目录为public，伪静态选择thinkphp
4、配置完成后，访问你的首页，即可开始安装。
5、安装完成后，后台地址是：http://你的域名/admin
